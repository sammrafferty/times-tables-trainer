# Times — Multiplication Rep Trainer

A fast, quantguidance-style drill trainer for learning the times tables up to 12 × 12. Single self-contained HTML file — no build step, works offline, installable on phone and desktop.

## Modes
- **Practice** — adaptive mixed drill that surfaces the facts you're slow or wrong on more often.
- **Focus** — grind a single table start to finish.
- **Speed** — 30/60/120-second sprint; shows your reps-per-minute pace.
- **Progress** — 12×12 mastery heatmap plus lifetime accuracy and average recall time.

## How it works
- Type the answer — it auto-advances the instant you're correct (no Enter).
- Tracks **speed**, not just accuracy: a fact counts as mastered only once recall is automatic (~2s).
- Progress is saved locally on the device.

## Run it
Just open `index.html` in any browser, or visit the deployed Vercel URL.
